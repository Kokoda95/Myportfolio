Evaluate a Manufacturing Process

This project focused on enhancing the monitoring and control of a manufacturing process by implementing Statistical Process Control (SPC). SPC uses data-driven methods to ensure the process operates within an acceptable range, defined by an Upper Control Limit (UCL) and Lower Control Limit (LCL). By analyzing historical manufacturing data using SQL, the goal was to define these limits and identify any deviations that require adjustments. This approach aims to maintain a smooth, high-quality production process.
